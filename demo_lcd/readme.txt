Example description
This examples demostrates the use of the LCD for basic graphics operations.
The Manley board LCD driver and the example are derived from the Manley LCD
demo code on the CDROM included with the Manley board.

The example shows simple use of Window, draw, and font functions.

Note: For a more comprehensive example, see the Manley board software.

Special connection requirements
There are no special connection requirements for this example.


Supported boards and board setup:
 LPCOPEN_11U6X_BOARD_MANLEY_11U68
